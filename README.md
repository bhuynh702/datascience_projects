# Data Science Projects

A collection of data science coursework, labs, and projects from IS 296 (Data Science) featuring exploratory data analysis, statistical modeling, hypothesis testing, and personal data analytics with Python and Jupyter Notebooks.

## Projects

### Personal Informatics
- **`Personal_Informatics/PI_Analysis.ipynb`** – Analysis of six weeks of self-tracked daily habits (exercise, study time, reading, sleep, and mood) to uncover patterns between lifestyle choices and emotional well-being.
- **`Personal_Informatics/lab12/`** – Permutation testing applied to personal data, with worked examples.
- **`Personal_Informatics/lab13/`** – Estimation techniques and confidence intervals.

### Sleep Health & Lifestyle Project
- **`project/Sleep_health_and_lifestyle_dataset.csv`** – Dataset of 374 records with 13 attributes (sleep duration, quality, physical activity, stress, BMI, blood pressure, and more) for analyzing sleep health patterns.

### Homework
- **`hw03/`** – Multi-dataset analysis covering Boston and Manila ride-sharing data, U.S. Census population estimates, and quarterly unemployment trends (1994–2024). Includes automated grading via Otter-Grader.

## Tech Stack

- **Language:** Python 3
- **Notebooks:** Jupyter / Google Colab
- **Libraries:** `datascience`, `numpy`, `matplotlib`
- **Testing:** Otter-Grader (automated assignment tests in `hw03/tests/` and `lab10/tests/`)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bhuynh702/datascience_projects.git
   ```
2. Install dependencies:
   ```bash
   pip install datascience numpy matplotlib jupyter
   ```
3. Launch Jupyter and open any `.ipynb` file:
   ```bash
   jupyter notebook
   ```

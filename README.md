# Data Science Projects

A collection of data science coursework, labs, and personal projects featuring data analysis and statistical modeling with Python and Jupyter Notebooks.

## Projects

### Personal Informatics
- **`Personal_Informatics/PI_Analysis.ipynb`** – Analysis of self-tracked personal data.
- **`Personal_Informatics/lab12/`** – Permutation testing examples.
- **`Personal_Informatics/lab13/`** – Estimation techniques.

### Homework
- **`hw03/`** – Data analysis with Boston housing, Manila, and U.S. demographic datasets.

### Labs
- **`lab04.ipynb`** – Lab 04 notebook.
- **`Lab09/`** – Lab 09 (PDF).
- **`lab10/`** – Statistical analysis with salary, player, and text data.
- **`lab11-assessing-model-template(stress).ipynb`** – Assessing models (stress dataset).

### Demos
- **`demo-assessing-models.ipynb`** – Model assessment demo.
- **`demo-week6-group.ipynb`** – Week 6 group demo.

### Data
- **`project/Sleep_health_and_lifestyle_dataset.csv`** – Sleep health and lifestyle dataset.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bhuynh702/datascience_projects.git
   ```
2. Install [Jupyter Notebook](https://jupyter.org/install) (or JupyterLab).
3. Open any `.ipynb` file to explore the analysis.
